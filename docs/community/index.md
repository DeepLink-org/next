# 社区

DeepLink 是浦江实验室发起的开源项目，致力于推动国产芯片跨域异构混训与超智融合技术的发展。

## 合作伙伴

DeepLink 的核心合作伙伴包括：

- **浦江实验室** — 项目发起方，核心研发
- **国家人工智能训练场（上海）** — 阶段一落地场景
- **无锡超算中心** — 阶段二超算互联试点
- **华为昇腾** — 国产芯片生态
- **寒武纪** — 国产芯片生态

## 学术合作

DeepLink 欢迎学术界合作。研究方向包括：

- 分布式训练拓扑优化
- 长距拥塞控制算法
- 异构芯片统一算子抽象
- 可重构组网架构
- AI for Science 应用
