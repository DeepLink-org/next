# 关于我们

DeepLink Next 由**浦江实验室 DeepLink 团队**发起，联合学术界与产业界共同推进。我们的目标是构建 AI for Science 的算力基座——这不是一家之事，需要芯片、超算、软件、科学应用多方力量共同参与。

---

## 从 DeepLink 到 DeepLink Next

DeepLink Next 不是凭空而来。它的起点是 **DeepLink 团队**过去三年在跨域异构混训上的持续突破。

2023 年，团队面对的问题是：国产 AI 芯片百花齐放，但各自为战——华为昇腾、寒武纪、昆仑芯之间无法在同一训练任务中协同。团队选择了最难的路径：不做硬件绑定，不做单一芯片生态，而是用软件在通用网络上抹平跨域、跨芯片、跨中心的差异。

三年后，DeepLink 团队在通用 RoCE 网络上跑通了**万卡规模、千公里距离、3+ 款国产芯片同任务训练**。MFU 达到英伟达同期主流方案的 80%，有效训练时间超过 95%。这套系统已在**国家人工智能训练场（上海）**投产，成为国内首个全国产万卡级跨域混训平台。

DeepLink Next 是在这个地基上往下一阶段走。从跨域互联到超融合，从任务切片到 Agent 运行时，从适配芯片到定义芯片——每一步都是同一个团队、同一个技术栈的自然延伸。

**DeepLink 是起点。DeepLink Next 是下一程。**

## 合作层级

不同的参与深度对应不同的投入、不同的署名权。我们明确每一层的对等关系。

| 层级 | 你投入 | 你获得 | 适合谁 |
|------|------|------|------|
| **联合编著** | 技术贡献 + 专家评审 + 联合发布 | 白皮书 / 技术规范联合署名 | 高校、研究机构、标准组织 |
| **联合实验室** | 算力资源 + 联研团队 + 场景数据 | 共建实验室挂牌 + 联合成果署名 + 研究生联培 | 超算中心、芯片厂商、科学计算团队 |
| **战略顾问** | 学术背书 + 方向指导 + 资源引荐 | 顾问委员会署名 + 重大成果致谢 | 学科带头人、资深学者 |
| **资源合作** | 算力 / 芯片 / 网络等基础设施 | 项目致谢 + 合作方 Logo 展示 | 硬件厂商、云服务商、数据中心 |
| **学术合作** | 研究方向共建 + 研究生联合培养 | 联合论文 / 联合课题署名 | 高校课题组、独立研究者 |
| **开源贡献** | 代码贡献 + 社区治理 | 项目 Committer / Maintainer 署名 | 个人开发者、开源社区 |

战略顾问不涉及日常研发投入，以学术声誉和方向判断为核心贡献。联合实验室适合愿意投入团队和算力的深度参与者。两个层级可以并行。
