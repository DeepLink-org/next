# AllSpark

大规模预训练框架，面向万亿参数模型的分布式训练。

## 命名

"AllSpark" 源自变形金刚中的火种源——塞伯坦的生命之源与终极能量。以此为预训练框架命名，寓意它是 DeepLink Next 所有模型的"火种"起点。

## 定位

AllSpark 是 DeepLink Next 的预训练基座，专为跨域异构集群设计。它在任务智能切片和长距通信库之上，提供完整的预训练工作流。

- 拓扑感知的模型分片与并行策略
- 跨域异构芯片的统一训练入口
- 弹性容错与自动恢复
- 生成的模型权重可直接被 NexRL 加载用于强化学习微调
