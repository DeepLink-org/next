# 项目收录

DeepLink Next 是一个开放的算力基础设施项目，欢迎与 AI4S 相关的开源项目申请收录。收录项目将获得生态入口、技术指导与社区资源支持。

---

## 收录范围

我们关注以下方向的开源项目：

| 方向 | 说明 | 示例 |
|------|------|------|
| 分布式训练与推理 | 训练框架、推理引擎、模型服务化 | AllSpark / Energon / Teletraan |
| 智能体与运行时 | Agent 框架、分布式通信、任务调度 | Pulsing / Persisting |
| 科学计算与仿真 | 科学算子库、PDE 求解器、分子动力学 | — |
| 算力调度与网络 | 跨域通信、拥塞控制、拓扑优化 | 长距通信库 |
| 可观测性与调试 | 分布式 profiling、tracing、日志 | Probing |
| 硬件与芯片工具链 | 芯片适配、算子编译、FPGA 工具 | — |

超出以上范围但与 AI4S 基础设施相关的项目，仍可提交申请，由技术委员会个案评估。

---

## 孵化等级

收录项目按成熟度分三级：

| 等级 | 状态 | 要求 |
|:---:|------|------|
| :material-seed-outline: **沙箱** | 早期项目，概念验证阶段 | 清晰的 README + 可运行的 demo |
| :material-sprout-outline: **孵化** | 活跃开发，有早期用户 | 开放治理 + 持续集成 + 用户文档 |
| :material-tree-outline: **毕业** | 生产可用，独立社区 | 稳定 API + 安全审计 + 多维护者 |

所有收录项目初始均为沙箱等级。升级由技术委员会按季度评审。

## 申请流程

```
提交申请 → 技术委员会初审（2 周）→ 公开评议（2 周）→ 收录决定
```

申请材料需包含：

- 项目名称与简介
- 与 DeepLink Next 生态的关系（解决了什么问题、与已有项目如何协作）
- 当前代码仓库地址（需开源，Apache 2.0 或兼容协议）
- 核心维护者信息
- 近期路线图

[:material-email-outline: 提交申请](mailto:deeplink@pjlab.org.cn){ .md-button }
