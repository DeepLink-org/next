# Teletraan

LLM 推理服务平台，面向生产级模型部署与 Agent 服务化。

## 命名

"Teletraan" 源自变形金刚中 Autobots 的中央超级计算机 Teletraan-1，负责管理、调度和监控所有 Autobots 的行动——与我们将其定位为推理服务的中央调度平台高度契合。

## 定位

Teletraan 是 DeepLink Next 的推理服务化层，基于 Matrix 推理框架，提供生产级的模型部署、调度和监控能力。

- OpenAI 兼容 API 网关
- 多模型、多版本路由与金丝雀发布
- 自动扩缩容与负载均衡
- 与 Pulsing 集成，将推理服务注册为分布式 Actor
- 与 Probing 集成，提供零侵入的性能监控
