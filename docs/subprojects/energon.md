# Energon

高性能 LLM 推理框架，面向异构芯片的大规模模型推理。

## 命名

"Energon" 源自变形金刚中驱动一切的能量源——正如 Energon 为 Autobots 提供动力，Energon 推理引擎为 AI 模型提供高效推理能力。

## 定位

Energon 是 DeepLink Next 的推理引擎层，为上层 Teletraan 推理服务平台和 Pulsing Agent 运行时提供高性能推理能力。

- 支持国产异构芯片的高性能推理后端
- FP16 / BF16 / INT8 / INT4 混合精度推理
- 连续批处理（Continuous Batching）与动态批合并
- 与 Persisting 集成，支持 KV Cache 持久化复用
