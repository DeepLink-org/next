# Probing

零侵入分布式调试器，面向 AI 工作负载的动态性能分析。

无需修改代码、无需重启进程，attach 即可用标准 SQL 查询性能数据。

- 零侵入注入（`probing -t <pid> inject`）
- SQL 驱动分析（Apache DataFusion）
- 动态代码执行与栈捕获
- 分布式跨节点关联
- <5% 性能开销
